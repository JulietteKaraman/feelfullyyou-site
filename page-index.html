How to Deploy

Option A. Netlify drag & drop (easiest)

  1. Go to netlify.com and log in
  2. Click Add new site → Deploy manually
  3. Drag the entire feelfullyyou-site folder onto the page
  4. Netlify gives you a live URL in seconds
  5. Go to Site settings → Custom domains to connect feelfullyyou.com

Option B. GitHub + Netlify (for ongoing edits)

  1. Create a new repo at github.com called feelfullyyou-site
  2. Upload the entire feelfullyyou-site folder into it
  3. In Netlify: Add new site → Import from Git
  4. Connect the GitHub repo. Publish directory = / (root)
  5. Every push to GitHub auto-deploys the live site

Plain HTML files, no build step, no framework, no terminal needed. Just drag, drop, done.
